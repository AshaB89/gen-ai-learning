# Experiments

Quick experiments, prototypes, and proof-of-concepts.

## Purpose

This directory is for:
- Rapid prototyping
- Testing new ideas
- Quick experiments with new libraries
- Learning new techniques
- Exploratory data analysis

## Naming Convention

Use descriptive names with dates:
- `2024-01-15_transformer_attention_visualization`
- `2024-01-16_gan_face_generation`
- `2024-01-17_bert_fine_tuning_experiment`

## Structure

Each experiment should be self-contained but can reference shared utilities.
